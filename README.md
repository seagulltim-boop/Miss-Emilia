# Miss Emilia

Static one-page website prototype for Miss Emilia.

## Local preview

Run a simple local server from this folder:

```powershell
py -3.12 -m http.server 4173 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4173/
```

Hero alternatives are available at:

```text
http://127.0.0.1:4173/hero-previews.html
```
